# Justice Frontend (Vite + React + Tailwind + Axios)

## Requirements
- Node.js 18+

## Setup
```bash
npm install
npm run dev
```

## Config API URL
Edit `.env`:
- `VITE_API_BASE_URL=http://127.0.0.1:8000`
